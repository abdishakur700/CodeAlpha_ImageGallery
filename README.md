# CodeAlpha_ImageGallery

## Project Overview

This project is a responsive image gallery created using HTML, CSS, and JavaScript. It allows users to view images in a lightbox, navigate between images, and filter images by category.

## Features

* Responsive image grid
* Lightbox image viewer
* Next and previous navigation
* Hover animations
* Category filtering
* Smooth transitions

## Technologies Used

* HTML5
* CSS3
* JavaScript
* typescript

## How to Run

1. Clone or download the repository.
2. Open the terminal
3. Install dependencies: `npm install`
4. Run the app:  `npm run dev`
